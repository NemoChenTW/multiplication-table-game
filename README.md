# Multiplication Table Game (九九乘法特訓班)

A fun and interactive web-based game to practice multiplication tables, designed especially for Taiwanese users with full Traditional Chinese support.

## Features

- **Interactive Quiz Mode**: Answer multiplication questions against the clock
- **Customizable Tables**: Select which multiplication tables (1-9) to practice
- **Real-time Timer**: Visual progress bar shows remaining time for each question
- **Performance Tracking**: Tracks correct and incorrect answers with detailed feedback
- **Wrong Answer Review**: See a list of questions you got wrong at the end
- **Responsive Design**: Works on desktop, tablet, and mobile devices
- **Instant Feedback**: Visual and audio feedback for correct/incorrect answers

## How to Use

1. Open `multiplication-table-game.html` in your web browser
2. Select the multiplication tables you want to practice (1-9)
3. Click "開始特訓" (Start Practice) to begin
4. Answer each question before time runs out
5. Review your results and see which questions you missed

## Game Rules

- Each question has a time limit to answer
- Correct answers are highlighted in green
- Incorrect answers are highlighted in red
- At the end, you'll see your total score and a list of wrong answers
- Click the restart button to practice again

## Browser Support

- Firefox

- HTML5

## Installation

No installation required! Simply download the file and open it in your browser.

```bash
git clone https://github.com/NemoChenTW/multiplication-table-game.git
cd multiplication-table-game
# Open multiplication-table-game.html in your browser
```
